# FamishQuest

## Descripción
FamishQuest es un videojuego de mazmorras con jugabilidad roguelike, donde el jugador deberá contratar mercenarios y aventureros para explorar las temidas y peligrosas mazmorras de “La muerte y Locura”. A medida que avanzas, encontrarás enemigos, tesoros y desafíos aleatorios que pondrán a prueba tu estrategia y habilidades.

## Características
- Generación aleatoria de mazmorras y recompensas.
- Combate estratégico basado en turnos.
- Gestión de equipo y recursos.
- Sistema de cordura y cansancio que afecta el rendimiento de los personajes.

- Sistema operativo: Windows, macOS, Linux

## Instalación
1. Clona el repositorio: git clone https://github.com/IvanDobobuto/FamishQuest.git
2. Navega al directorio del proyecto: FamishQuest
3. Compila el proyecto: make

## Uso
- Ejecuta el juego desde la terminal: ./FamishQuest
- Navega a través de las mazmorras, gestiona tu equipo y sobrevive a los desafíos.
